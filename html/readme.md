### 画像関係

PC-SP間で画像を出しわける事があると思います。

その際はdisplayで出しわけるのではなく、pictureタグを使って切り替えるようにしてください。

またRetina対応の為、スマホ画面で画像がぼやける場合は、スマホ用は従来の2倍の大きさで画像を書き出してpictureタグで出し分けてください。
